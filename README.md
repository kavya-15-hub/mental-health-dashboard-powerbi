# Mental Health Analytics Dashboard (Power BI)

## Overview
This project analyzes user mental health using key metrics such as stress, anxiety, mood, sleep, and burnout.

## Tools Used
- Power BI
- DAX
- Excel

## Key Features
- KPI-based wellness scoring system
- Interactive dashboard with multiple health indicators
- Visual breakdown of stress, mood, sleep, and burnout factors

## Insights
The analysis shows moderate stress and anxiety levels, while sleep quality remains relatively high. Burnout indicators highlight the need for better stress management.
